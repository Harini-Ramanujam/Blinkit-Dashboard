# Blinkit Sales Dashboard - Power BI

## Project Overview
This project is an interactive Power BI dashboard created to analyze Blinkit's sales performance, outlet growth, customer ratings, and product insights. The dashboard helps in understanding business trends and making data-driven decisions.

## Key Metrics
- Total Sales: $1.20M
- Average Sales: $141
- Number of Items: 8523
- Average Rating: 3.9

## Dashboard Features
- Outlet establishment trend analysis
- Sales by outlet size
- Sales by outlet location
- Product item type analysis
- Fat content comparison
- Outlet-wise performance summary
- Dynamic filters for easy analysis

## Tools Used
- Microsoft Power BI
- Excel
- Data Cleaning
- Data Visualization
- DAX

## Business Insights
- Tier 3 outlets generated highest sales
- Medium size outlets performed best
- Supermarket Type1 had highest revenue
- Product categories showed balanced ratings
